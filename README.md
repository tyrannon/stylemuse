
# 👗 StyleMuse

**StyleMuse** is your AI-powered virtual closet. Upload your clothing, generate smart outfit suggestions using GPT-4o Vision, and curate your personal style—all from your phone.

---

## ✨ Features

- 📸 **Wardrobe Upload**  
  Snap or upload photos of your clothes to build a digital wardrobe.

- 🧠 **AI Outfit Generator**  
  StyleMuse uses GPT-4o to generate outfit ideas based on your actual clothing.

- 🖼️ **Gallery View**  
  Browse, tap, and swipe through your saved outfit ideas.

- 🔍 **Item Detail Modal**  
  Tap on a saved item to view it full screen, delete it, or swipe down to dismiss.

---

## 🛠 Tech Stack

- **React Native** (Expo)

## 📦 Setup

1. Clone the repo  
   ```bash
   git clone https://github.com/yourusername/stylemuse.git
   cd stylemuse


2. Install dependencies

   ```bash
   npm install
   ```
3. Run the app

   ```bash
   npx expo start
   ```

> You'll need an OpenAI API key to enable AI-powered outfit generation.
